Cấu trúc tài liệu

<html>
Thẻ gốc của tài liệu HTML, bao bọc toàn bộ nội dung.
<!DOCTYPE html>
<html lang="vi">
  <head>
    <meta charset="UTF-8">
    <title>Tiêu đề</title>
  </head>
  <body>
    Nội dung trang
  </body>
</html>
(Không hiển thị trực tiếp — xem mô tả)
<head>
Phần đầu tài liệu: chứa metadata, tiêu đề, liên kết CSS/JS.
<head>
  <meta charset="UTF-8">
  <title>Tiêu đề</title>
  <link rel="stylesheet" href="style.css">
</head>
(Không hiển thị trực tiếp — xem mô tả)
<meta>
Metadata cho tài liệu (charset, viewport, SEO).
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
(Không hiển thị trực tiếp — xem mô tả)
<title>
Tiêu đề trang (hiện trên tab trình duyệt, SEO).
<title>Trang web của tôi</title>
(Không hiển thị trực tiếp — xem mô tả)
<base>
Đặt URL gốc cho tất cả liên kết tương đối.
<base href="https://example.com/">
(Không hiển thị trực tiếp — xem mô tả)
<link>
Liên kết tài nguyên ngoài (CSS, icon...).
<link rel="stylesheet" href="style.css">
(Không hiển thị trực tiếp — xem mô tả)
<style>
Chèn CSS nội bộ trong tài liệu.
<style>
  p { color: blue; }
</style>
(Không hiển thị trực tiếp — xem mô tả)
<script>
Chèn JavaScript (không hiển thị trực tiếp).
<script>
  console.log('Xin chào từ JS');
</script>
(Không hiển thị trực tiếp — xem mô tả)
<body>
Phần nội dung hiển thị cho người dùng.
<body>
  <h1>Xin chào!</h1>
</body>
(Không hiển thị trực tiếp — xem mô tả)

<b>

<del>

<ins>

<code>

Ví dụ: <div> là khối.
